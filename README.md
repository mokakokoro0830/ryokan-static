# 月詠 -Tsukuyomi-

Premium ryokan landing page — a fictional 8-room hidden inn nestled in the mountains of Shinshu, est. 1923.

> 🌐 **Live**: https://mokakokoro0830.github.io/ryokan-static/

## About

A portfolio piece exploring high-end Japanese hospitality on the web. Vertical Japanese typography, parallax onsen scenes, and brand-specific interactive features that go beyond a typical ryokan site.

## Features

- 🌙 **Moon Phase Calendar Reservation** — Pick a date, see the moon phase that night. Auto-suggests the next full moon.
- 📜 **Vertical Haiku Telop** — Side-pinned vertical-rl haiku that fades between sections as you scroll
- 🍂 **Seasonal Garden Switcher** — Spring / Summer / Autumn / Winter tabs that swap image, copy, and accent color
- 💎 **Custom Dropdown** — Black-and-gold styled select instead of native `<select>`
- 🎌 **Bilingual JP / EN** — Japanese-primary with subtle English subtitles for international guests

## Tech Stack

- Pure **HTML / CSS / JavaScript** — no frameworks, no build step
- Fonts: Cormorant Garamond / Shippori Mincho / Noto Serif JP
- Images: AI-generated via OpenAI gpt-image-1 + Unsplash
- Responsive (PC / tablet / smartphone)
